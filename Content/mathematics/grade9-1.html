Example 1: Linear Function

A linear function is a function of the form y = mx + b, where m is the slope and b is the y-intercept.

Consider the following linear function:
f(x) = 3x + 5

To explicitly demonstrate this function:

1. Domain: The domain of this function is the set of all real numbers, as x can take on any real value.

2. Range: The range of this function is also the set of all real numbers, as y can take on any real value for different values of x.

3. Mapping: For any input value x, the function maps it to the output value y = 3x + 5. For example:
   - When x = 0, f(0) = 3(0) + 5 = 5
   - When x = 2, f(2) = 3(2) + 5 = 11
   - When x = -1, f(-1) = 3(-1) + 5 = 2

The graph of this linear function is a straight line with a slope of 3 and a y-intercept of 5.

Example 2: Quadratic Function

A quadratic function is a function of the form y = ax^2 + bx + c, where a, b, and c are constants.

Consider the following quadratic function:
g(x) = 2x^2 - 3x + 4

To explicitly demonstrate this function:

1. Domain: The domain of this function is the set of all real numbers, as x can take on any real value.

2. Range: The range of this function is a set of real numbers that depends on the values of the constants a, b, and c. In this case, the range is also the set of all real numbers.

3. Mapping: For any input value x, the function maps it to the output value y = 2x^2 - 3x + 4. For example:
   - When x = 0, g(0) = 2(0)^2 - 3(0) + 4 = 4
   - When x = 2, g(2) = 2(2)^2 - 3(2) + 4 = 8 - 6 + 4 = 6
   - When x = -1, g(-1) = 2(-1)^2 - 3(-1) + 4 = 2 + 3 + 4 = 9

The graph of this quadratic function is a parabola with a vertex at the point (-3/4, 7/4).

These examples demonstrate the explicit definitions and mappings of linear and quadratic functions, which are two fundamental types of mathematical functions with widespread applications in various fields.
